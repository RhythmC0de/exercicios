# Cadastro de Doadores de Sangue


Este projeto é um **script simples em Python** que simula um cadastro básico de doadores de sangue por meio do terminal.

O programa coleta informações do usuário, calcula a idade e verifica se a pessoa atende aos **requisitos mínimos de peso e idade** para doação.

---

## Funcionalidades


* Solicita os seguintes dados:

  * Nome completo
  * Peso (kg)
  * Altura (cm)
  * Ano de nascimento
* Calcula a idade com base no ano informado
* Verifica:

  * Peso mínimo para doação
  * Idade mínima para doação
* Exibe um resumo formatado com os dados do doador

---

## Requisitos


* Python 3.x

---

## Como executar


No terminal, dentro da pasta do projeto:

```bash
python index.py
```

Em seguida, preencha as informações solicitadas.
